# Pygame Game Project

このプロジェクトはPygameを使用したゲーム開発のテンプレートです。

## セットアップ方法

1. Pythonがインストールされていることを確認してください
2. 必要なパッケージをインストール:
   ```
   pip install -r requirements.txt
   ```
3. ゲームを実行:
   ```
   python main.py
   ```

## 操作方法

- ウィンドウを閉じる: ウィンドウの×ボタンをクリック

## 開発について

このプロジェクトは基本的なPygameの構造を提供しています。`main.py`を編集して、ゲームの機能を追加してください。
